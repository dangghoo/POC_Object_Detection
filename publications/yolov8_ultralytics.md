# YOLOv8 — Ultralytics

**Role in this project:** Object detection model — fine-tuned YOLOv8-nano (`yolov8n.pt`) to detect the pineapple cup in shelf photos. The detected bounding box centre is mapped to a 3×3 grid cell to determine shelf position.
